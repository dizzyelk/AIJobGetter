# AIJobGetter - 自动投递简历脚本

AIJobGetter 是一个基于 Python 和 `DrissionPage` 开发的自动化网页操作脚本，主要用于在招聘平台（目前支持 Boss 直聘）上自动浏览并投递符合条件的职位。

它可以帮你解放双手，在设定好投递条件和目标数量后，自动完成繁琐的“打招呼”和投递动作。

## 🌟 支持的功能 (Features)

目前版本已实现以下核心功能：

1. **半自动安全登录**：脚本启动后会打开浏览器并暂停，等待用户手动完成安全登录操作，避免触发风控，登录完成后按回车即可继续任务。
2. **基于经验要求的自动筛选**：内置了经验关键词过滤（如 `'在校/应届'`, `'经验不限'`, `'1年以内'`, `'1-3年'`），只向符合经验要求的职位投递简历。
3. **全自动批量投递**：自动在推荐职位列表中滚动、识别，匹配成功后自动点击“立即沟通”并发起会话。
4. **投递数量控制**：支持自定义最大投递数量（默认 10 个），达到目标后自动停止，防止触发平台限制。
5. **进度实时打印**：在控制台实时输出当前正在投递的公司名称、职位名称、地点以及已投递的数量。

## 🚀 快速开始 (Quick Start)

### 1. 环境准备
确保你的电脑上安装了 Python 环境（建议 Python 3.8+）。

安装必需的依赖库：
```bash
pip install DrissionPage
```

### 2. 投递前配置 (重要)

为了让脚本投递得更精准，建议你在 Boss 平台上提前配置好你的求职意向和打招呼语：

* **设置投递职位、城市和薪资**：
  1. 登录 Boss 直聘网页版，在“推荐职位”页面顶部找到你的**求职期望**栏。
  2. 点击编辑或新建求职期望，设置好你想要投递的**职位名称**、**城市**以及**薪资要求**。
  3. 选中该求职期望，确保页面下方展示的是你想投递的职位列表。
* **设置打招呼语**：
  1. 建议在 Boss 直聘手机 App 端进行设置。
  2. 进入 **我的 -> 右上角设置图标 -> 常用语 -> 招呼语**。
  3. 添加一段能够展现你优势的打招呼语，并将其设置为**默认使用**。
  4. 这样脚本在自动点击“立即沟通”时，系统就会自动发送这段开场白。

### 3. 运行脚本
直接运行 `main.py` 文件：
```bash
python auto_apply.py
```

### 4. 操作步骤
1. 脚本会自动打开浏览器并跳转到 Boss 直聘的职位列表页。
2. **请在浏览器中手动完成登录**（扫码或密码登录）。
3. 登录完成后，回到命令行窗口，**按下回车键**。
4. 脚本将开始自动获取职位列表并根据条件进行投递，你可以去喝杯咖啡了！☕
